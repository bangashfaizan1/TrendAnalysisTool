In order to test our code. Your system should have node and npm package installed.

1. go to the root directory and 'run npm install' command which will install the dependencies.
2. go to the client directory and 'run npm install' command which will install the dependencies.
3. 'run npm dev' to start the port and once the server is started.
4. go to localhost:3000 to view the webapp.
5. Once clicked on the trend tab, click enter in the url so it can redirect you back to search page.
